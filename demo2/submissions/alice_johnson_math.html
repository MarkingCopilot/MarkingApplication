Problem 1: Find the derivative of f(x) = 3x² + 2x - 5

Solution:

f'(x) = d/dx(3x²) + d/dx(2x) - d/dx(5)

f'(x) = 6x + 2 - 0

f'(x) = 6x + 2

Problem 2: Evaluate ∫(4x³ - 2x + 1)dx

Solution:

∫(4x³ - 2x + 1)dx = ∫4x³dx - ∫2xdx + ∫1dx

= 4(x⁴/4) - 2(x²/2) + x + C

= x⁴ - x² + x + C

Problem 3: Find the limit: lim(x→2) (x² - 4)/(x - 2)

Solution:

lim(x→2) (x² - 4)/(x - 2)

= lim(x→2) (x + 2)(x - 2)/(x - 2)

= lim(x→2) (x + 2)

= 2 + 2 = 4
